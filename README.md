# Momento Store (Updated)
هذا المشروع متجر إلكتروني بسيط لبيع الملابس الرجالية والشبابية — تم تحديثه لتحسين التصميم، استبدال المنتجات لتتناسب مع الملابس، وإضافة تعليمات نشر على GitHub Pages.

## ما تم تغييره
- استبدال مجموعة المنتجات بملابس رجالية وشبابية (عناوين بالعربية والإنجليزية).
- تحسين واجهة العرض باستخدام Tailwind + CSS مخصص.
- إضافة تعليمات النشر (GitHub Pages).

## نشر على GitHub Pages
1. أنشئ مستودعًا جديدًا على GitHub وادفع (push) ملفات المشروع إليه.
2. في إعدادات المستودع (Settings) > Pages، اختر الفرع `main` أو `master` ومجلد `/ (root)`، ثم اضغط Save.
3. بعد لحظات سيكون الموقع متاحًا على `https://<your-username>.github.io/<repo-name>/`.



## الميزات المنفذة الآن
- واجهة متجاوبة محسّنة للموبايل والكمبيوتر.
- فلترة حسب الفئة، المقاس، واللون، وفرز بالأسعار.
- صفحة تفاصيل منتج (مودال) مع معرض صور.
- سلة جانبية قابلة للتعديل وحفظ في localStorage.
- صفحة دفع مع تحقق مبسط ومسح السلة بعد الطلب.
